Portland Bike to Transit Map

This handy map can display bike routes, transit stops, and transit routes at the same time, so you can find your own way to bike to transit stops.

Hover over or click on stops and routes for more info. If you like this, you might also like another project I'm working on, TriMet's Regional Trip Planner (beta), which can plan biking and bike-to-transit trips. Visit my website for more maps.

The original version of this site was awarded "Most Original App" in the second round of Portland's CivicApps contest. Thank you! And thank you also to the awesome teams that work on MapBox, TileMill and Wax, who provided me the opportunity to finally get this project up and running again!

Sources: OpenStreetMap (bike routes from 7/14/12, basemap styled by MapBox), transit data from TriMet. See any errors? Let me know! I can edit OSM or do an update: saxbarm at gmail dot com / twitter @pdxmele. Thanks!
